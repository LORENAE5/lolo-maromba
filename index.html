Entendendo RGB e CMYK

  • RGB
  • CMYK

RGB - Red, Green, Blue

O modelo de cores RGB é utilizado principalmente para exibição de cores em dispositivos como monitores e telas de celulares. Nesse modelo, as cores são formadas pela combinação de luzes vermelha (Red), verde (Green) e azul (Blue).

Essas cores podem ser combinadas em diferentes intensidades para criar uma vasta gama de cores. O valor de cada cor varia de 0 a 255, sendo 0 a ausência de luz e 255 a intensidade máxima.

O modelo aditivo de cores, como o RGB, é um método de criação de cores a partir da combinação de luzes de diferentes cores primárias. No caso do RGB, essas cores primárias são vermelho (Red), verde (Green) e azul (Blue). O princípio básico do modelo aditivo é que, ao combinar luzes dessas cores em diferentes intensidades, conseguimos criar uma ampla gama de outras cores.

CMYK - Cyan, Magenta, Yellow, Key (Black)

O modelo de cores CMYK é utilizado principalmente em impressão. Ele combina ciano (Cyan), magenta (Magenta), amarelo (Yellow) e preto (Key) para criar cores no papel.

Esse modelo trabalha com a absorção de luz, sendo que cada tinta absorve uma cor diferente e reflete as outras. As porcentagens de cada cor determinam a tonalidade final.

O modelo subtrativo de cores, como o CMYK, é usado principalmente para impressão e funciona de forma oposta ao modelo aditivo (RGB). Enquanto o RGB cria cores a partir de luz, o CMYK trabalha com tintas ou pigmentos que absorvem (subtraem) certos comprimentos de onda de luz e refletem outros. Esse processo é baseado na ideia de que a cor final que vemos é determinada pela luz que é refletida pelas tintas e não pela luz que é emitida.
